## 太鼓達人-網頁板 插件
這是一套使用插件擴展 [太鼓達人-網頁板](https://github.com/bui/taiko-web) 的功能集。將這些 .taikoweb.js 檔案存儲到一個資料夾中（在下面的鏈接上點擊郵件另存新檔），並透過自定義歌曲列表的按鈕在遊戲中載入他們。

### 遊戲玩法
- [**改變音樂速度 Change Music Speed**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/change-music-speed.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/change-music-speed.taikoweb.js)
	加快或減慢音樂在遊戲中的撥放速度

- [**變更時間窗口 Change Timing Window**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/change-timing-window.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/change-timing-window.taikoweb.js)
	自定義遊戲內音符的輸入間隔

- [**自定義滾動速度 Custom Scroll Speed**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/custom-scroll-speed.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/custom-scroll-speed.taikoweb.js)
	改變音符在遊戲中滾動的速度

- [**彩虹皇冠 Rainbow Crown**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/rainbow-crown.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/rainbow-crown.taikoweb.js)
	Donder-Full 連擊將會給你彩虹皇冠

- [**在多人遊戲中跳過遊戲結算 Skip Results in Multiplayer**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/skip-results-in-multiplayer.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/gameplay/skip-results-in-multiplayer.taikoweb.js)
	啓用在多人遊戲中跳過遊戲結算的屏幕，然而其他沒有安裝插件的玩家將無法看到完整的遊戲結算畫面

	- [使用者程式碼版本](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/userscripts/skip-results-in-multiplayer.user.js)

### 自定義歌曲
- [**Green Notes**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/green-notes.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/green-notes.taikoweb.js)
  Adds support for green notes (G) and ad-lib notes (F) in custom charts

- [**自定義小節線 Custom Barlines**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/custom-barlines.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/custom-barlines.taikoweb.js)
  Adds #BARLINESCROLL and #BARLINE to the tja format. #BARLINESCROLL can be used to set independent speed values on the measure lines, can be set to a floating point value or off. #BARLINE can insert measure lines anywhere in the chart.

- [**載入背景 Loading Background**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/loading-background.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/loading-background.taikoweb.js)
	新增「#BARLINESCROLL」和「#BARLINE」到 tja 格式。#BARLINESCROLL 可以設定在指定的測量線內設定獨立的速度數值，可以設置成小數點或是關閉（off）。#BARLINE 可以插入測量線到譜面中的任何位置

- [**Fumen 檔案格式 Fumen File Format**](https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/fumen-file-format.taikoweb.js) [(⬇️)](https://minhaskamal.github.io/DownGit/#/home?url=https://github.com/KatieFrogs/taiko-web-plugins/raw/main/custom-songs/fumen-file-format.taikoweb.js)
	爲自定義歌曲列表支援使用 Fumen 檔案格式
